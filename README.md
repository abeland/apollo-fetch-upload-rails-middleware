# ApolloFetchUploadRailsMiddleware

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'apollo_fetch_upload_rails_middleware'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install apollo_fetch_upload_rails_middleware
